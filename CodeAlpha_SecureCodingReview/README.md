# Python Secure Code Auditor

## Description
This project scans Python source code for common security vulnerabilities.

## Features
- Detect hardcoded passwords
- Detect SQL injection patterns
- Detect dangerous eval() usage
- Detect weak exception handling
- Generate vulnerability reports

## Technologies Used
- Python
- Regex
- File Handling

## Project Structure

├── assets/
├── sample_code/
├── reports/
├── scanner.py
├── rules.py
├── README.md

## How to Run
```bash
python scanner.py
```
