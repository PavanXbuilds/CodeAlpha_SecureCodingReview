## 📖 About
A **static analysis tool** that parses Python source files and flags dangerous coding
patterns before they reach production. Inspired by real-world SAST principles.

## ✨ Features
- 🔑 **Hardcoded Passwords** — `CRITICAL`
- 💉 **SQL Injection Patterns** — `CRITICAL`
- ⚠️ **Dangerous `eval()` Usage** — `HIGH`
- 🐛 **Weak Exception Handling** — `MEDIUM`
- 📄 **Generates Vulnerability Reports**
